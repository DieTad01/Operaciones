El método gráfico es una herramienta clave en la investigación de operaciones que permite representar gráficamente problemas de programación lineal. Se utiliza principalmente para problemas que involucran dos variables de decisión, facilitando la visualización de las restricciones y la función objetivo. Este método ayuda a encontrar la región factible, que es el conjunto de soluciones que cumplen con todas las restricciones del problema.

Notas

  • Soporta restricciones con ≤, <=, ≥, >= y = (internamente maneja iguales como dos desigualdades).
  • Siempre se incluye x ≥ 0 y y ≥ 0.
  • El óptimo se busca evaluando la función objetivo en todos los vértices de la región factible.
